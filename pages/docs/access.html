Access control

Configure role-based access control to ensure only authorized engineers can access specific sites.

Roles

  • Admin — Full access to all sites and settings
  • Engineer — Access to assigned sites, can initiate OOB sessions
  • Viewer — Read-only access to dashboard and logs

SSO integration

Equate supports SAML 2.0 and OAuth 2.0 for single sign-on integration with your identity provider.
