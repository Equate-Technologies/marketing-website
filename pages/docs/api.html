API reference

The Equate API provides programmatic access to your fleet data and OOB management capabilities.

Authentication

API requests require a valid API key. Generate keys from Settings → API Keys.

bash 
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.equate.io/v1/sites

Endpoints

  • GET /v1/sites — List all sites
  • GET /v1/sites/:id — Get site details
  • POST /v1/sites/:id/sessions — Initiate OOB session
  • GET /v1/audit — Retrieve audit logs
