platform

The infrastructure behind every connection.

Equate is built from four interlocking layers. Together, they give your team eyes and hands at every remote site — even when the internet's gone.

Request early access See how it works
hardware

The device that never gives up.

A small hardware appliance sits at each remote location. It connects directly to on-site routers and switches via serial and console cables — the same low-level access technicians use. Unlike software-only solutions, the physical device never depends on the local network's health.

RS-232 serial · RJ-45 console
Serial / Console
Cellular LTE
agent

Lightweight. Persistent. Self-healing.

The Equate agent is a tiny service on the device. It immediately opens an outbound-only tunnel to Equate’s cloud over cellular — no inbound firewall rules, no VPNs. The agent auto-updates, self-heals, and keeps your engineers connected no matter what.

Heartbeats every 15 seconds
Signed heartbeat packets stream to Equate. Miss three in a row and your alerts fire instantly.
Outbound-only
The agent never listens on a port. All sessions are broker-initiated from the cloud — your firewall stays untouched.
Zero-maintenance
Install once. The cloud auto-updates the agent. No manual patching or nightly jobs.
Encrypted tunnel
TLS 1.3 plus authenticated session layer. The broker validates both ends before terminal data flows.
agent@eq-tx2-01 ~ equate-agent
cloud

Your infrastructure's always-on backbone.

Equate’s cloud runs on AWS serverless, scaling with your fleet. It brokers every session between engineers and agents. Session content is never stored — the cloud simply relays encrypted traffic and orchestrates alerts.

AWS broker.equate.io · Alert engine + IoT Core

dashboard

Every site. One screen.

The Equate dashboard shows every managed location, status, device count, and last heartbeat. Click “Connect” to open a browser terminal without any VPN. Alerts fire the moment something starts to slip.
