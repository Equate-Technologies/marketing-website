Introduction

Equate is an out-of-band (OOB) network management platform that gives IT teams secure access to distributed infrastructure — even when the primary network is completely unreachable.

ℹ
Equate is currently in early access. Some features described here may not be available in all plans. See pricing →

What is out-of-band management?

Traditional network management relies on the same network it's managing. When that network goes down, you lose visibility and control of your devices. Out-of-band management uses a completely independent path — typically a cellular connection — so you can always get in, regardless of primary network state.

How Equate works

Each site runs a lightweight Equate agent on a supported device (Raspberry Pi, cellular router, etc.). The agent maintains a persistent connection to the Equate cloud backend over cellular, independent of your primary WAN. When your primary link fails, the agent detects it, Equate alerts your team, and you connect via the dashboard for a secure terminal session to any device at that site.

Quickstart

Get your first site online in under 30 minutes.

Prerequisites

  • A supported hardware device (see Supported hardware)
  • An active SIM card with data from any major carrier
  • An Equate account — request early access

Step 1: Install the agent

SSH into your device and run the install script:

bash 
curl -fsSL https://install.equate.io/agent | sudo bash

Step 2: Authenticate the agent

After install, run the auth command with your site token from the dashboard:

bash 
equate-agent auth --token YOUR_SITE_TOKEN

Step 3: Verify connectivity

Check the dashboard. Your site should appear with a green "Online" status within 60 seconds.

✓
Once the agent shows Online, your OOB path is active. The agent will automatically failover when your primary link drops.
