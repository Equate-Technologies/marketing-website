Monitoring platform

Evidence from the network
to the dashboard.

The platform keeps customer-network access local while giving operators durable, cloud-backed monitoring state.

Local SNMP collection
A customer-side collector polls configured SNMPv2c devices independently, with core interface data and supported Cisco and Arista enrichment.
Durable outbound delivery
Telemetry is written to a local SQLite outbox before MQTT/TLS delivery, so a transport outage does not stop polling.
Cloud-owned monitoring state
Ingestion validates and deduplicates telemetry before PostgreSQL becomes the source of truth for dashboard and API views.
Dependency-aware health
Separate direct Critical devices from Unknown devices affected by unavailable upstream paths—without treating impact as proof of an outage.
